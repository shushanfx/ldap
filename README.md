# Gradle构建工具
Gradle构建工具是针对VR开发的一系列构建工具，包括两个部分：pc(pc_vr)和无线(wap_vr)。  
两者主要是配置不一致以及包含部分不一致的代码实现，但是构建流程保持一致。  

## pc_vr
````
  pc端构建工具：      
	* 初始化; 
	* 生成dev版本; 
	* 生成test版本;
	* 生成online版本;
	* 合并至trunk;
	* 预览;
	* 模板校验;
````

## wap_vr
````
  无线端构建工具： 
    * 初始化; 
    * 生成dev版本; 
	* 生成test版本;
	* 生成online版本;
	* 合并至trunk;
	* 预览;
	* 模板校验;
````
